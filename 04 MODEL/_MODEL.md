# Conceptual MODEL Folder
This folder contains the **models** for whatever the project has to deal with: environment, users, workers, stakeholders, interactions, data, APIs, protocols, and finally data and datastructures (schema) etc.  

Further this folder contains the central repository for business terms (glossary), objects and references and links to real live business workflows, processes, activities, resources, plans, documents, and other artifacts.

Goal of this folder is to contain and describe all project and product/service related elements representing real world objects.
